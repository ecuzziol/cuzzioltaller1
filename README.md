## Sobre este proyecto

- Este proyecto fue desarrollado en PHP 8.4
- La version de node usada es la 25.9
- La version de Laravel es 5.25



Para correr este proyecto es necesario correr primero

- composer install
- npm run build (para compilar todos los assets de scss css y js asi como para agregar las dependencias de npm y de vite)
- puedes importar el proyecto en laravel Herd 1.28 y correrlo desde ahi o bien 
- luego corre "php artisan serve" en una terminal 
- luego corre "npm run dev" en otra terminal
